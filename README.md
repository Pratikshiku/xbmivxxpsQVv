### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

2. 学生管理
3. 学生列表
4. 宿舍管理

9. 安全退出
